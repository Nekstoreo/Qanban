## 🔧 Instalación

```bash
# Clona el repositorio
git clone https://github.com/Nekstoreo/Qanban.git

# Navega al directorio
cd Qanban

# Abre la aplicación (macOS)
open index.html

# Alternativa para Linux
xdg-open index.html

# Alternativa para Windows
start index.html
```
